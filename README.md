# Quiz do Biel 🎯

Um quiz interativo com três níveis de dificuldade e efeitos visuais modernos.

## 📱 Demo

O quiz está disponível online e pode ser acessado através do navegador. Desenvolvido com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- 3 níveis de dificuldade:
  - 🟢 **Fácil**: Perguntas básicas de conhecimentos gerais
  - 🟡 **Médio**: Perguntas intermediárias sobre ciência, história e geografia
  - 🔴 **Difícil**: Perguntas avançadas sobre matemática, física e química

- Cada nível contém 5 perguntas diferentes
- Feedback visual imediato para respostas certas e erradas
- Animações e efeitos visuais modernos
- Design responsivo
- Pontuação final ao término do quiz

## 🎨 Design e Interface

- Botões com gradientes modernos
- Animações suaves de transição
- Efeitos de hover e clique
- Feedback visual com cores (verde para acerto, vermelho para erro)
- Layout limpo e intuitivo

## 🚀 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Escolha um dos três níveis de dificuldade
3. Responda as 5 perguntas do nível escolhido
4. Veja sua pontuação final
5. Opção de jogar novamente com outra dificuldade

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com animações e transições)
- JavaScript (vanilla)
- Font Awesome para ícones

## 🎮 Como Jogar

1. **Escolha a Dificuldade**
   - Fácil: Para iniciantes
   - Médio: Para quem busca um desafio moderado
   - Difícil: Para os experts!

2. **Durante o Quiz**
   - Leia a pergunta com atenção
   - Clique na resposta que você acha correta
   - Feedback imediato mostrará se você acertou ou errou
   - A resposta correta será destacada em verde

3. **Pontuação**
   - Cada resposta correta vale 1 ponto
   - Pontuação máxima: 5 pontos por nível
   - Veja seu resultado final ao término do quiz
